阿里云部署web项目

2020-10-29
字数统计: 798   |   阅读时长≈ 3 分钟

1.购买阿里云

  • 登陆阿里云

  • 搜索云翼计划 购买学生机==9.9/月==

  • 购买后记住ip 配置root密码

  • 在服务器中选择操作系统 可选ubuntu16

  • 在本地远程登陆阿里云 可使用自带xshell工具或者putty第三方工具

    • 输入阿里云账号 端口号默认22

    • 登陆成功

#2.配置用户环境

1.修改主机名

hostnamectl set-hostname ‘ubuntu’

然后去阿里云控制台重启服务器。

2.创建非root用户

创建普通用户：briup/briup

3.给新用户授予权限

vim /etc/sudoers

4.切换briup用户登陆

3.搭建JavaWeb运行环境

与在本地启动项目一样，需要安装JDK环境，tomcat（服务器）以及mysql

1.安装JDK

  • apt在线安装

    openjdk

    (1)查找合适的openjdk版本:

    ​ apt-cache search openjdk

    (2)安装

    sudo apt-get install openjdk-8-jdk

    (3) 配置环境变量, 编辑如下文件

    vim ~/.bashrc

    在最后加上一行

    (4) 测试jdk是否安装成功:

    java -version

    显示如下结果, 则说明安装成功:

    ##oraclejdk

  • 官网下载

    ==注：apt下载oracle需要仓库==

    （1）将文件放置在briup用户下

    ==ubuntu中需要安装SSHServer==

    sudo apt install openssh-server

    方法一：shell工具

    方法二：filezillla

    （2）解压jdk的包

    tar -xvf jdk...

    （3）配置环境变量 （家目录）

    vi.bashrc

    (4) 使配置资源生效

    source .bashrc

2.安装tomcat

  • 官网下载

同上，将安装包移到服务器即可

3.安装mysql

  • 在线安装

    1>安装

    apt-get install mysql-server-5.7

    2>开启mysql远程访问

    select user,host from user

    可看到host全是localhost，

    localhost表示只能本地登录，可以将root账号对应的host改为%即可实现远程连接（暂不考虑安全性）

    update user set host = '%' where user = 'root'

    3>设置mysql编码

    vim /etc/mysql/my.cnf

    4>重启mysql服务

    service mysql restart

    //service mysql status 查看mysql状态

    5>在线下载mysql

  • 官网下载

4.部署war包

1>将web项目打成war包

2>将war包放入服务器上tomcat/webapps下面

3>开启服务器,会自动解压war包

4>需要的话，修改默认端口号

#5.在阿里云控制台上开放相关端口号

比如8080端口，3306或者你的项目端口，否则无法访问。
